Taller de Páginas Web con IA

Cómo trabajamos

En este taller el docente trabaja con una IA que se llama Claude Code.

Claude Code no es el Claude del navegador. Es una versión especial que tiene acceso a la computadora del docente. Puede crear carpetas, crear archivos, instalar cosas, y subir código a internet. Todo en tiempo real.

Cuando el docente le da una instrucción — por ejemplo "creá un archivo HTML" — Claude Code lo crea, lo sube al servidor, y en segundos aparece en esta página.

Eso que van a ver hoy es exactamente cómo trabaja un desarrollador profesional.

Cómo enviamos mails desde Claude Code

Claude Code puede enviar mails desde Gmail directamente desde el código, usando Python. No abre el navegador — se conecta a Gmail con una clave especial.

Esa clave se llama contraseña de aplicación. No es tu contraseña normal de Gmail. Es una clave de 16 caracteres que Google genera específicamente para que programas externos puedan enviar mails en tu nombre.

Para obtenerla hay que ir a:

https://myaccount.google.com/apppasswords

Requisito: la cuenta de Google tiene que tener activada la verificación en dos pasos.

Una vez obtenida, la clave se guarda en un archivo llamado .env en la computadora. Ese archivo nunca se sube a internet — está protegido por el .gitignore.

Cómo interactuar con Git

Git es el sistema que guarda el historial de cambios del proyecto. Cada vez que modificamos un archivo, Git puede registrar ese cambio con un mensaje que explica qué se hizo.

Desde Claude Code, Git se usa con tres comandos principales:

1. git add — le decís a Git qué archivos querés guardar

2. git commit — guardás los cambios con un mensaje descriptivo

3. git push — subís los cambios a GitHub, y desde ahí se despliegan automáticamente al servidor

En este taller, cada vez que el docente dice "subilo", Claude Code ejecuta esos tres comandos y en segundos el cambio aparece en la página que están viendo.

No hace falta escribir los comandos a mano. Claude Code los ejecuta solo cuando recibe la instrucción.
